# qssvec

**qssvec** is a Python package for approximate simulation of quantum circuits based on a sparse and truncated state vector representation, with a particular focus on peaked circuits.

This repository is associated to the paper *A Sparse and Truncated State Vector Simulator for Peaked Circuits*,  
submitted to the 2026 IEEE International Conference on Quantum Computing and Engineering (QCE26), also known as [IEEE Quantum Week 2026](https://qce.quantum.ieee.org/2026/).
