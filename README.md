# qssvec

**qssvec** is a Python package for approximate simulation of quantum circuits based on a sparse and truncated state vector representation, with a particular focus on peaked circuits.

This repository is associated to the paper *A Sparse and Truncated State Vector Simulator for Peaked Circuits*, submitted to the 7th IEEE International Conference on Quantum Computing and Engineering (QCE26), often referred to as the [IEEE Quantum Week 2026](https://qce.quantum.ieee.org/2026/).
